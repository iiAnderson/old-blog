My Projects

These projects are designed to push my boundaries into using new technologies and development practices.

Biometric Authentication for Cloud Services

While there are many secure authentication mechanisms avaliable to us, users often choose the least secure due to the easy use. In this project I set out to design and develop a system which could be both secure and easy to use. The project encompass cutting edge research into the anonymisation of biometrics, the development of scalable & secure software platforms to achieve this goal. Furthermore, I integrated the project with Google Authenticator - allowing it to be used as a wrapper around two factor authentication measures, something which has not been developed before.
